# Terramo

> A basic Carbon Footprint Awareness Platform designed to help users understand and reduce their environmental impact. Created for the Google PromptWars Hackathon.

## Features

* **Carbon Calculator:** Estimate your personal carbon emissions based on basic daily habits.
* **Eco-Insights:** Read actionable tips and recommendations to lower your footprint.
* **Simple Interface:** A clean, easy-to-navigate frontend design.

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript

## Getting Started

Since this is a lightweight, frontend-only project, there is no complex installation or backend setup required. 

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/terramo.git](https://github.com/your-username/terramo.git)
   ```

2. Navigate into the project folder:
   ```bash
   cd terramo
   ```

3. Open the `index.html` file directly in any web browser to view the application.

## Usage

1. Open the website in your browser.
2. Input your basic daily activities into the calculator.
3. View your estimated carbon footprint and read the suggested tips to improve your daily habits.
